Project Overview

This comprehensive DFIR investigation simulates a real-world security incident where an enterprise network was compromised. The project demonstrates end-to-end incident response capabilities including initial detection, evidence acquisition, memory forensics, disk analysis, network traffic examination, and detailed timeline reconstruction of attacker activities.

Investigation Objectives

Initial Detection

Identify indicators of compromise through log analysis and anomaly detection

Memory Analysis

Extract running processes, network connections, and malicious artifacts from RAM

Disk Forensics

Recover deleted files, analyze file system artifacts, and examine persistence mechanisms

Timeline Reconstruction

Build comprehensive attack timeline from initial access to data exfiltration

Investigation Methodology

1

Evidence Acquisition

Acquired memory dump using FTK Imager and created forensic images of compromised systems following proper chain of custody procedures

2

Memory Forensics

Analyzed memory dump with Volatility 3 to identify malicious processes, injected code, and network connections

vol.py -f memory.dmp windows.pslist
vol.py -f memory.dmp windows.netscan
vol.py -f memory.dmp windows.malfind
3

Disk Analysis

Used Autopsy to examine file system, recover deleted files, and analyze browser history, registry entries, and user activities

4

Network Analysis

Examined PCAP files with Wireshark to identify C2 communications, data exfiltration attempts, and lateral movement

5

Timeline Creation

Correlated artifacts from multiple sources to build comprehensive attack timeline using SIFT Workstation tools

Key Findings

Critical

Initial Access Vector

Spear-phishing email with malicious macro-enabled document. Victim opened attachment leading to execution of PowerShell payload.

High

Persistence Mechanism

Registry Run key modification and scheduled task creation for maintaining access. Identified through registry analysis and Windows event logs.

High

Lateral Movement

Pass-the-hash attacks using compromised credentials. Network logs revealed RDP connections to multiple internal systems.

Critical

Data Exfiltration

Sensitive documents compressed and exfiltrated via HTTPS to external IP. Total of 2.3 GB of data transferred over 48-hour period.

Tools & Techniques

Memory forensics framework for extracting digital artifacts from RAM dumps

Autopsy

  • File system timeline generation

  • Keyword searching and data carving

Wireshark

Network protocol analyzer for traffic inspection

  • PCAP file analysis

SIFT Workstation

Ubuntu-based forensics distribution with pre-installed tools

  • Timeline analysis with log2timeline
  • File system examination
  • Memory analysis utilities

Lessons Learned

  • Chain of Custody: Maintaining proper documentation and evidence handling procedures is critical for forensic integrity
  • Multi-Source Correlation: Combining evidence from memory, disk, and network sources provides comprehensive incident understanding
  • Timeline Reconstruction: Accurate timeline creation requires careful correlation of timestamps across different artifact types
  • Tool Proficiency: Deep understanding of forensic tools and their capabilities is essential for effective investigation
  • Documentation: Detailed note-taking and screenshot capture throughout the investigation supports accurate reporting

Conclusion

This DFIR investigation demonstrated comprehensive incident response capabilities from initial detection through forensic analysis and timeline reconstruction. The project successfully identified the attack vector, documented persistence mechanisms, traced lateral movement, and confirmed data exfiltration activities. The findings were compiled into a detailed forensic report suitable for legal proceedings and organizational security improvements.

The investigation reinforced the importance of systematic evidence collection, proper tool usage, and thorough documentation in digital forensics work. These skills are directly applicable to real-world SOC and incident response scenarios.
