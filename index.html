Category Digital Forensics & Incident Response
Duration 3 Weeks
Tools Volatility, Autopsy, Wireshark, SIFT

Project Overview

This comprehensive DFIR investigation simulates a real-world security incident, covering the complete investigation lifecycle from initial detection through forensic analysis to final reporting. The project demonstrates practical skills in memory forensics, disk analysis, network traffic investigation, and timeline reconstruction.

Investigation Phases

01

Initial Detection

Identified suspicious system behavior through anomalous network connections and unusual process execution patterns in system logs.

02

Memory Forensics

Analyzed memory dump using Volatility framework to extract running processes, network connections, and loaded drivers.

03

Disk Analysis

Performed deep disk forensics with Autopsy to recover deleted files, analyze file system artifacts, and identify malicious payloads.

04

Network Investigation

Examined network traffic captures using Wireshark to identify command and control communications and data exfiltration attempts.

Key Findings

Malicious Process Injection

Discovered process hollowing technique used to inject malicious code into legitimate system process (svchost.exe)

Command & Control Communication

Identified encrypted C2 traffic to external IP addresses using custom protocol over port 443

Data Exfiltration

Detected unauthorized data transfer of sensitive documents totaling 250MB to attacker-controlled server

Persistence Mechanism

Uncovered registry modifications and scheduled tasks created for maintaining persistent access
