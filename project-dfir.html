Project Overview

This comprehensive DFIR investigation lab demonstrates a complete incident response workflow, from initial detection to timeline reconstruction and threat attribution. The case study simulates a real-world security incident involving malware infection and data exfiltration.

Investigation Methodology

  • Initial detection through SIEM alert correlation
  • Memory dump acquisition and analysis using Volatility
  • Disk forensics with Autopsy for artifact recovery
  • Network traffic analysis using Wireshark
  • Timeline reconstruction and IOC identification
  • Malware reverse engineering and behavioral analysis

Key Findings

  • Identified APT-style malware with command and control communication
  • Recovered 15+ indicators of compromise (IOCs)
  • Mapped attack to MITRE ATT&CK framework (T1566, T1059, T1041)
  • Documented complete attack timeline spanning 72 hours
  • Discovered evidence of lateral movement and privilege escalation

Tools & Technologies

  • Volatility: Memory forensics and process analysis
  • Autopsy: Disk forensics and file recovery
  • Wireshark: Network traffic analysis and PCAP inspection
  • FTK Imager: Forensic image acquisition
  • YARA: Malware signature detection
  • Splunk: Log correlation and analysis

Lessons Learned

This investigation highlighted the importance of comprehensive log collection, proper evidence preservation, and systematic analysis methodology. The case demonstrated how combining multiple forensic techniques provides a complete picture of the attack lifecycle.
