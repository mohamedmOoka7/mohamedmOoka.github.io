DFIR Investigation Lab

Incident Response & Forensics Case Study

1. Project Overview

This project simulates a real-world security incident requiring a full DFIR investigation. The objective was to detect malicious activity, analyze forensic artifacts, and reconstruct the attacker’s timeline.

2. Incident Detection

The incident was initially detected through abnormal authentication behavior identified in Windows Security Event Logs. Multiple failed login attempts were followed by a successful authentication from an unusual source.

  • Event ID 4625 – Failed logon attempts
  • Event ID 4624 – Successful logon
  • SIEM correlation rule triggered

3. Tools & Technologies

Splunk SIEM Windows Security Logs Sysmon PowerShell Timeline Analysis

4. Investigation & Analysis

  1. Reviewed Windows Security logs for abnormal authentication activity
  2. Identified multiple failed logons followed by a successful login
  3. Analyzed Sysmon logs to detect suspicious PowerShell execution
  4. Correlated events to reconstruct the full attack timeline

6. Log Evidence

6.1 Failed Login Attempts – Event ID 4625

Analysis of Windows Security logs revealed repeated failed authentication attempts from the same source, indicating a brute-force attack.

Figure 1 – Windows Security Event ID 4625 showing failed logon attempts

6.2 Successful Login – Event ID 4624

A successful login occurred shortly after multiple failures, confirming credential compromise.

Figure 2 – Windows Security Event ID 4624 showing successful authentication

6.3 PowerShell Execution – Sysmon Event ID 1

Sysmon logs showed an encoded PowerShell command execution, a common post-exploitation technique.

Figure 3 – Encoded PowerShell execution detected in Sysmon logs

6.4 Privilege Escalation Attempt

Further investigation revealed suspicious activity consistent with a privilege escalation attempt.

Figure 4 – Evidence of privilege escalation attempt

7. Results & Outcome

The investigation successfully identified the attack vector, reconstructed the timeline, and extracted relevant indicators of compromise. Recommendations were provided to improve detection and response capabilities.
