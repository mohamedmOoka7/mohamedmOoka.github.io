DFIR Investigation Lab

Incident Response & Forensics Case Study

Project Overview

This project simulates a real-world security incident requiring full DFIR investigation. The goal was to identify attacker activity, collect evidence, and build an incident timeline.

Incident Detection

The incident was initially detected through abnormal authentication activity identified in Windows Security Event Logs. Multiple failed login attempts were followed by a successful authentication from an unusual source.

  • Event ID 4625 – Failed logon attempts
  • Event ID 4624 – Successful logon
  • SIEM correlation rule triggered

Tools & Technologies

SIEM Windows Event Logs Linux Timeline Analysis Memory Forensics

Investigation & Analysis

Log analysis revealed suspicious authentication attempts followed by privilege escalation. Artifacts were collected and correlated to reconstruct attacker behavior.

Results & Outcome

The investigation successfully identified the attack vector, timeline, and indicators of compromise (IOCs). Recommendations were provided to improve detection and response capabilities.
